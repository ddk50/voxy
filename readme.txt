
VoXY

◆確認環境
　・OS: Gentoo Linux (32bit版)
　・ブラウザ: Firefox 3.6.13
　・ライブラリ:
　　・Cherrypy 3.1.2 (Gentoo 標準)
　　・twisted 9.0.0 (Gentoo 標準)

◆インストール方法
　・ディストリビューションのレポジトリからtwistedとcherrypyをインストールする
　・targets.cfg.exampleを参考に、設定ファイルをtargets.cfgという名前で作成する

◆実行方法
　・voxy.pyを実行する
　・voxy.pyを実行したホストのポート8012に対してWebブラウザからアクセスする
　　・URLは、例えば対象ホスト名がvoxyhostなら「http://voxyhost:8012/」
